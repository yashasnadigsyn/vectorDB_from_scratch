## VectorDB from Scratch

Trying to write and compare different vector database implementations from scratch

Currently implemented:
- Flat search (Brute force)
- IVF 
- IVF Optimized with K means Plus Plus and other optimizations
- No dependency IVF from scratch using built in python libraries
- NSW
- HNSW with no neighbor heuristic
- HNSW with neighbor heuristic
